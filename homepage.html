Undefined is global variable (window) with primitive type of undefined. It is returned when:

  • variable is defined but not assigned value of type
  • checking typeof of undeclared variable
  • accessing non-existing property of object
  • function argument has been omitted in a call
  • function does not explicitly return a value

Idea is to be a computed (returned) value.

The value null represents the intentional absence of any object value.

It is never returned by the compiler.

Calling typeof on a null value returns "object".
